# Banking Management System


## Description
Banking management system using C. It's a toy project for learning C deeply and advanced C. You can run this program on your own machine, add feature and fix bugs.


## Features
- Create account
- Login
- User Menu
- User Profile
- Diposit Money
- Transfer Money
- Logout
- Exit
